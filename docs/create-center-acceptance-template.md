# 创作中心验收清单模板

## 覆盖任务

- `G-020` 创作中心验收清单模板
- `C-015` 创作中心基础冒烟测试脚本

## 通用验收清单

| 检查项 | 期望 |
| --- | --- |
| 路由可访问 | `/create-center` 和 11 个 `/create-center/:stepId` 页面均可打开 |
| 项目创建 | 空项目状态下可创建新项目并进入步骤一 |
| 项目切换 | 顶部切换项目按钮可切换当前项目 |
| 步骤导航 | 每个步骤可点击跳转，当前步骤高亮 |
| 状态反馈 | 加载、错误、保存、生成均有可见反馈 |
| 步骤一保存 | 填写核心故事后可保存并刷新回显 |
| 文件导入 | `.txt`、`.md`、`.docx` 入口存在，导入后显示文件名与解析状态 |
| 响应式布局 | 窄屏不横向溢出 |

## 冒烟脚本建议

当前可用 `npm run build:web` 做静态构建门禁。后续接入浏览器自动化后，建议新增 `npm run smoke:create-center`，覆盖项目创建、步骤跳转和步骤一保存回显。
