# TEST-DOC-001：11 步创作中心冒烟测试与验收清单

## 文档目标

本文档用于 coMGan-ai 创作中心 11 步页面逐步落地后的统一冒烟测试与验收。验收重点覆盖：

- 创作中心骨架：项目创建、项目切换、路由、进度、最后修改时间。
- 11 步页面：路由可达、保存、刷新回显、未保存提示、错误提示、空状态、下一步跳转。
- 跨步骤数据流：上一步产物能否被下一步正确读取和复用。
- 真实用户全链路门禁：新建项目 -> 输入/导入 -> 生成 -> 编辑 -> 保存 -> 关闭再打开 -> 继续编辑 -> 删除/归档。

标注说明：

- `自动化`：适合后续用 Playwright、API 脚本或单元测试稳定覆盖。
- `人工`：需要人工视觉、交互、文案、可用性判断。
- `混合`：可自动断言基础结果，但仍建议人工确认视觉或流程体验。

## 真实用户全链路门禁

每次主流程较大改动后，必须以真实用户身份跑以下门禁链路。

| 编号 | 动作 | 类型 | 通过标准 |
| --- | --- | --- | --- |
| GATE-001 | 新建项目 | 自动化 | 项目创建成功并进入创作中心 |
| GATE-002 | 步骤 01 输入核心故事和集数 | 自动化 | 保存成功，刷新回显 |
| GATE-003 | 步骤 02 导入或输入文本 | 自动化 | 文本可保存，剧本可编辑 |
| GATE-004 | 使用任一生成入口生成草稿 | 混合 | 生成结果可展示、可编辑、可保存 |
| GATE-005 | 编辑生成结果 | 自动化 | 编辑后的结果覆盖展示为最新内容 |
| GATE-006 | 保存当前步骤 | 自动化 | 保存成功提示明确 |
| GATE-007 | 关闭页面后重新打开项目 | 自动化 | 项目仍存在，最后编辑内容回显 |
| GATE-008 | 继续编辑后续步骤 | 自动化 | 上一步数据可被下一步读取 |
| GATE-009 | 模拟保存失败 | 自动化 | 错误提示明确，用户输入不丢失 |
| GATE-010 | 切换到另一个项目再切回 | 自动化 | 两个项目数据隔离 |
| GATE-011 | 删除或归档项目 | 混合 | 删除或归档后列表状态正确，详情页不再误打开 |
| GATE-012 | 移动端宽度抽查 | 人工 | 关键流程可读、可点、可操作 |

门禁结论必须只允许：

- `通过`
- `不通过`

若任意必检项不通过，不允许对外宣告该功能已完成。

## 自动化与人工检查边界

### 优先自动化的验收点

- 路由可达性。
- 项目创建、删除、切换。
- 保存接口成功与失败。
- 刷新回显和关闭再打开回显。
- 当前步骤、当前进度、最后修改时间。
- 非法 projectId、非法 stepId。
- 列表项增删改。
- 跨步骤数据是否存在。
- 任务状态枚举展示。

### 必须人工视觉/交互检查的验收点

- 页面视觉是否符合黑金主题和液态玻璃风格。
- Hover、选中态、边缘荧光、按钮位置是否符合需求。
- 字幕遮挡、安全区、视频预览体验。
- 生成结果是否语义合理。
- 空状态文案是否清楚。
- 移动端布局是否拥挤、重叠、误触。
- 长文本阅读与编辑体验是否顺畅。

### 混合检查的验收点

- AI 生成入口：自动化可检查接口和状态，人工确认结果质量。
- 资产预览：自动化可检查元素存在，人工确认图片/视频/音频可用。
- 导出版本：自动化可检查记录，人工确认预览和命名合理。
- 复盘回流：自动化可检查数据结构，人工确认不会误覆盖前序创作成果。
