# 移动端步骤导航折叠方案

## 覆盖任务

- `G-019` 移动端步骤导航折叠方案

## 当前落地

创作中心在窄屏下已经将整体工作台从双栏布局降为单列布局，步骤导航保留在工作台上方区域，避免主内容被左右挤压。

## 后续增强

| 断点 | 导航形态 | 交互 |
| --- | --- | --- |
| `>1280px` | 左侧固定步骤列表 | 点击步骤路由跳转 |
| `768px-1280px` | 单列顶部步骤列表 | 纵向滚动 |
| `<768px` | 折叠横向步骤条 | 左右滑动，当前步骤居中 |
| `<520px` | 只显示编号和当前步骤名 | 点击展开完整步骤菜单 |

## 验收点

移动端不出现横向页面溢出；步骤入口可点击；当前步骤有高亮和状态圆点；键盘或触摸均能操作。
